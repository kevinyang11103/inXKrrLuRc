# 图书管理系统 python386

## 项目概述

图书管理系统是一个基于Vue与Python开发的现代图书管理解决方案。系统目前正处于更新优化阶段，致力于通过高效的技术架构，提供便捷的图书管理功能。

## 技术栈

- **前端**：使用 Vue2 进行构建
- **后端**：采用基于Python的 web 框架 FastAPI
- **可视化**：集成 Echarts 进行数据展示

## 功能模块

- 基础的图书管理功能
- 数据统计分析模块
  - 数据可视化分析
  - 数据可视化展示

## 系统角色

- 图书管理员
- 普通用户

- 前端：兼容现代浏览器的环境
